## 기획 의도 및 배경


**다시, 여행.**

100만, 그리고 1억.

2023년 1분기 공항버스 이용객의 숫자, 그리고 같은 시기 chatGPT의 월 이용자 숫자입니다.

엔데믹이 불러온 여행 붐, 그리고 첫 발을 내딛은 AGI, Auto-GPT.

우리는 사용자의 니즈와 기술적 고민을 한 데 담아냈습니다.

순간의 감상과 추억을 기록하고,

여행지에서 만난 소중한 인연과 추억을 나눠보세요.

여행에 필요한 모든 추천은 Yeo-Bot이 해줄거예요.

모여와 함께 여행을 떠나보아요.
